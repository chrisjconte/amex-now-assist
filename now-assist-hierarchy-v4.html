Prompt Management
Templates with dynamic variable injection via mustache syntax. Lock production prompts and clone for iteration.
Data Privacy for Now Assist
PII masking before prompts reach any LLM. OOTB patterns for SSNs, credit cards, emails, phone numbers, national IDs. Works with all providers.
Now Assist Guardian
Content moderation across 16 categories including toxicity, bias, and false advice. Opt-in per workflow — admins enable on configured skills.
LLM Provider Routing
Pre-built Integration Hub connectors per provider. Swap models at the skill, skill group, or instance level without code changes.

Skills and agents are written once. Which model executes them is a configuration decision managed through Now Assist Admin. PII masking, content filtering, and audit logging apply across all providers.

AI Agent Orchestrator
Coordinates multi-agent workflows. Manages handoffs between agents, resolves conflicts, and tracks state across long-running agentic tasks.
↑ Back to top

Runtime coordination layer managing how multiple agents collaborate — ensuring reliability and full auditability on every multi-agent execution.

Coordination
  • Task routing to specialist agents
  • Three planner strategies: Base (linear), ReActive (iterative), Batch (parallel with dependencies)
  • Context passing with full state preservation
Reliability
  • Error management and re-routing of failed tasks
  • Human-in-the-loop checkpoint management
  • Escalation triggers based on time, action count, or sentiment
Interoperability
  • MCP (Model Context Protocol) for external tool connections
  • A2A (Agent-to-Agent) for external AI integration
  • Full execution trace with plan, task, tool, and message-level logging
